Logo

Either you are keeping your site without logo, as is by default, or you upload a small logo and replace your shop name.
We recommend to use a small, standard-sized logo of 320×120 pixels. However you can freely set your logo width and height.


Color scheme

You can completely replace all colors by creating a new CSS file (Custom Color Scheme), or you can simply change the major colors of the current scheme here.
If you are skilled in CSS we recommend you to create a Custom Color Scheme.
