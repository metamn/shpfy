Snippets

Snippets are used to translate longer text and/or to set up external services like Twitter, Facebook, RSS Feeds.
In any case you must review / edit these snippets to customize theme defaults to your own settings.


Collections at Frontpage

Frontpage can display 3 collections of products: Featured, Latest and Sales.
It is highly recommended to use these three collections to better show off your products.

Featured products are enabled by default, Latest and Sales must be set up manually.
Instructions how to set up the Latest Products and Products on Sale you can find highlighted on your theme's frontpage.


Logo

Either you are keeping your site without logo, as is by default, or you upload a small logo and replace your shop name.
We recommend to use a small, standard-sized logo of 120Ã—60 pixels. However you can freely set your logo width and height.


Color scheme

You can completely replace all colors by selecting a Custom Color Scheme and then editing the CSS file (recommended for advanced users).
If you are not skilled in CSS you can simply change all colors manually aided by a color picker.

Translations

You can completely replace all texts the site uses to display messages.
You can replace the original English text, or translate theme to your own language.


Twitter, RSS Feed

Here you can set up external services embedded in your theme. If you are not using a service you can simply disable.
The RSS Feed displays the titles of your latest blog posts at the Frontpage, below your list of products in the footer.
Similarily there is a box in the footer displaying the avatars of your Twitter followers.


Mobile Version

This theme adapts itself to various screen sizes from large desktop displays to tablets, smartphones and small mobile screens.
The functionality and the design is fully preserved across all devices, there is no less content on smaller screens.
Mobile version was not tested on all existing devices only on Apple, Nokia, Samsung and RIM smartphones. If you consider not to use this feature you can switch it off below.
